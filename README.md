# Website Thefter

Инструмент для сохранения полной копии веб-страницы со всеми ресурсами на локальный диск. Позволяет просматривать сайт офлайн без подключения к интернету.

## Возможности

- Скачивает страницу целиком: HTML, CSS, JavaScript, изображения, шрифты, видео.
- Заменяет все ссылки на локальные пути
- Интерфейс на русском и английском языке.

## Как это рабоатет
- Вы копируете ссылку на нужный вам сайте
- Отправляете в нашу утилиту
- Через ~5-10 секунд получает готовую папке в корне
- Там заходите в папку также с названием сайта
- Запускаете .html файл

## Требования

- Python 3.8 или новее
- Установите зависимости:

```bash
pip install requests beautifulsoup4
```
## Установка и запуск
Важно: все файлы из архива должны лежать в одной папке
- Скачайте архив и распакуйте его.
  
 ```bash
git clone https://github.com/CtrlAltDeleted-Team/Website-Thefter.git
```

- Убедитесь, что все файлы находятся в одной папке, не разносите их по разным местам.

- Выполните
```bash
python main.py
```
